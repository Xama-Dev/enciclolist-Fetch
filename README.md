# Enciclolist-Fetch
### Aplicação que carregada uma lista de itens de acordo com o tema selecionado pelo usuário. As requisições ao servidor são realizadas via Fetch.

Tecnologias usadas: 
- HTML5
- CSS3: Grid layout.
- JavaScritp/EcmaScript 6: Clean code.
- Manipulação DOM
- AJAX, Fetch
- ExpressJs
- Nodemon
